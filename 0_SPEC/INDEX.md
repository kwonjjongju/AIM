### UI 특징
- **좌측 아이콘 바** (64px, 어두운 슬레이트): 대시보드, 개선보드, 관리 메뉴
- **서브메뉴 사이드바** (220px, 밝은 색): 선택된 메뉴의 하위 항목
- **상단 헤더** (56px): 시스템 타이틀 + 사용자 정보
- **메인 컨텐츠**: 선택된 페이지 표시
