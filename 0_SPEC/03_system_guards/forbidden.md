# 금지 행위 (Forbidden)
## 업무 개선 보드 시스템

---

## 1. 개요

금지 행위는 시스템 개발 및 운영에서 **절대 하면 안 되는 것**을 정의한다.
이 문서에 명시된 항목은 **예외 없이 준수**해야 한다.

---

## 8. 코드 리뷰 체크리스트

## 금지 사항 체크리스트

### 비즈니스
- [ ] KPI/점수/평가 기능 없음
- [ ] 부서 간 랭킹 없음
- [ ] 달성률/퍼센트 없음

### 보안
- [ ] 비밀번호 해싱됨
- [ ] JWT Secret 환경 변수
- [ ] 권한 검사 있음
- [ ] Raw SQL 없음

### 데이터
- [ ] 소프트 삭제 사용
- [ ] 트랜잭션 적용
- [ ] 이력 기록

### 코드
- [ ] .env 커밋 안됨
- [ ] 에러 처리됨
