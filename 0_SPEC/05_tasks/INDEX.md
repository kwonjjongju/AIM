## 빠른 참조

### ID 접두사
- `TASK-###`: 작업 ID
- `TASK-###-FE`: 프론트엔드 작업
- `TASK-###-BE`: 백엔드 작업

### 작업 상태
| 상태 | 설명 |
|------|------|
| TODO | 미시작 |
| IN_PROGRESS | 진행 중 |
| BLOCKED | 차단됨 |
| DONE | 완료 |

### Phase별 작업 수
| Phase | 작업 수 | 예상 기간 |
|-------|---------|-----------|
| Phase 1 (MVP) | 15개 | 2주 |
| Phase 2 | 6개 | 1주 |
| Phase 3 | 4개 | 1주 |
