## 입력 가이드

### 필수 입력 항목
- 부서명, 담당자명, 담당자 이메일
- 업무명, 업무 내용
- 주요 문제점, 개선 필요 사유
- 개발 목적
- 입력/출력 데이터 관련 항목

### 선택 입력 항목
- 기술/구현 방식 (아이디어 단계면 빈칸 가능)
- 인프라/운영 환경 세부사항
- 기타 제약사항

### 입력 팁
1. **업무 내용**: 가능한 구체적으로 기술
2. **주요 문제점**: 현재 Pain Point를 명확히
3. **기대 효과(정량)**: 시간/비용 절감 수치 제시
4. **입력/출력 데이터**: 개발 범위 판단에 매우 중요
