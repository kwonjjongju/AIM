## 4. 테스트 우선순위

- TC-005-01: 로그인/인증

- TC-002-01: 상태 변경
- TC-006-01: 권한 검사

### 4.2 P2 (권장)
- TC-003: 조회/필터링
- TC-004: 대시보드
